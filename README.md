# 🎙️ Master Transcriber Pro (Skill for Antigravity & VS Code)

Este repositório contém a Skill **Master Transcriber Pro**, desenvolvida pela **Oliveira Labs**. Esta skill transforma seu assistente de IA em um especialista em transcrição de áudio de alta performance.

## 🌟 O que esta Skill faz?
Ao ativar esta skill, a IA recebe instruções precisas para gerar uma aplicação Python completa com:
- **Interface Gráfica (GUI):** Visual profissional com branding Oliveira Labs.
- **Processamento Robusto:** Suporte para áudios longos (+1h30).
- **Exportação Inteligente:** Formatação automática para SRT (legendas) e TXT (texto limpo).
- **Facilidade de Uso:** Seletor de arquivos nativo do Windows.

## 🛠️ Como Instalar e Organizar

### 1. Uso Global (Sempre Disponível)
Para que a skill esteja acessível em qualquer projeto sem precisar ser copiada:
- **Caminho:** Salve o arquivo `SKILL_OLIVEIRA_LABS_MASTER_TRANSCRIBER.md` dentro da pasta de conhecimento central do seu assistente.
- **Localização Comum:** Geralmente encontrada em `%USERPROFILE%\.gemini\antigravity\knowledge\`.
- **Vantagem:** A IA terá acesso a esta "especialidade" em qualquer conversa global.

### 2. Uso Local (Pasta Invisível de Projeto)
Para que a skill atue apenas na pasta onde você está trabalhando, mantendo o ambiente limpo:
- **Caminho:** Na raiz do seu projeto, crie uma pasta invisível chamada `.gemini` (ou `.antigravity`) e salve o arquivo lá dentro.
  - Exemplo: `meu-projeto/.gemini/SKILL_OLIVEIRA_LABS_MASTER_TRANSCRIBER.md`
- **Vantagem:** O arquivo fica "escondido" de buscas comuns e da árvore de arquivos principal, mas o assistente de IA o lerá automaticamente como uma instrução de contexto para aquela pasta específica.

## 🚀 Como Ativar
Com o arquivo no local correto, basta solicitar:
> *"Ative a skill Master Transcriber Pro para processar este projeto."*
